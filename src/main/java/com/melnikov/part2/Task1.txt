    ООП - объектно-ориентруемое программирование - стиль программирования, при котором программные компноненты представляются
в виде объектов, а сама работа программы происходит благодаря взаимосвязи этих объектов между собой
    4 принципа ООП:
    1) Инкапсуляция - сокрытие реализации;
    2) Наследование - возможность одного класса наследовать поля и методы другого класса (родительского);
    3) Полиморфизм - один интерфейс - множество реализаций;
    4) Абстракция - реализация только того функционала, который необходим;
